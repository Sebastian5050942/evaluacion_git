# Evaluación Unidad 1

# Descripción del proceso:

- En esta clase de Evaluación creamos un nuevo Repositorio en el cual internamente creamos 3 archivos que tienen como nombre:

1. guia_local.md
2. guia_remoto.md
3. lectura_sensor.py

# Comandos utilizados para la creación de la primera parte:

- Al inicio se debe verificar con el comando "pwd" para ver nuestra ubicación, después de verificar lo anterior procedemos a entrar a nuestra carpeta la cual creamos en la primera clase, en mi caso "Esemargtcj", esto lo hacemos utilizando el comando "cd", cuando ingresamos creamos una carpeta llamada "evaluacion_git", al momento de crear esto debemos crear 3 archivos los cuales su nombre se pueden ver en la "Descripción del proceso" usando el comando "touch + nombre del archivo", antes de verificar que se hayan creado correctamente debemos crear el git., el cual utilizamos el comando "git init", Cuando ya hicimos lo anterior, debemos agregar los 3 archivos utilizando el comando "git add ." y verificamos con "git status" para que nos confirme si fue exitoso

# Comando exacto para crear el Repositorio:

- "git init"
